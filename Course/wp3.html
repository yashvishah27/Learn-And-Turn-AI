Problem?

Before we jump on to finding the algorithm for evaluating the problem and searching for the solution, we first need to define and formulate the problem.
Problem formulation involves deciding what actions and states to consider, given the goal.
For example, if the agent were to consider the action to be at the level of “move the left foot by one inch” or “turn the steering wheel by 1 degree left”, there would be too many steps for the agent to leave the parking lot, let alone to Bucharest. In general, we need to abstract the state details from the representation.
